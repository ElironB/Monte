# Monte Decision Lab

Repo-local React dashboard for showcasing Monte's persona model, simulation results, runtime telemetry, evidence loop, and source observations.

## Development

```bash
npm install
npm run dev
```

The app defaults to `http://localhost:3001` and targets the Fastify API on `http://localhost:3000`.

Override the API target with:

```bash
cp .env.example .env
```
